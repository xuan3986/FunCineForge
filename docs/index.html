Dataset Features

海量高清资源

数据集收录了上百部热门影视剧，经过 FunCineForge 工具自动化生产得到的数据集。

多模态标注

不仅包含文本字幕、音频、视频数据，还提供线索标注、说话人年龄性别音色标注、时间戳标注、类型标注、面部时序数据、唇部时序数据、音频特征、语义 Tokens 等多模态多类型数据。

Frequently Asked Questions

Q: 数据集如何解压？

A: 数据集下载后，请采用 7z 命令进行解压缩：

$ 7z x hongloumeng.7z.001

请使用 MD5 进行文件校验确认文件完整性：

$ md5sum -c hongloumeng.md5sum

Q: 包含哪些影视剧？

A: 数据集主要覆盖2010年至2025年的热门影视剧，同时也收录了部分经典老剧如《红楼梦》、《三国演义》等。

Q: 如何生产自己的数据集？

A: 您可以参考我们开源代码的 README 进行自动化高质量数据生产。为了保证模型效果，建议参考以下要求收集影视剧：

  • 视频来源：电视剧或电影，非纪录片，人物独白或对话场景较多，人脸清晰且无遮挡（如无面罩、面纱）。
  • 语音要求：发音标准，吐字清晰，人声突出。避免方言、背景噪音过大或口语感过强的素材。
  • 画面质量：高像素，面部细节清晰，光线充足，避开极端阴暗或强背光的画面场景。

Q: 如何反馈其他问题？

A: 您可以在 FunResearch Issues 中留言提问，或与 开发者 邮件联系。
